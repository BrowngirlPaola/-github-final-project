# Simple Interest Calculator

## Author
BrowngirlPaola

## Description
A simple interest calculator that computes interest based on user input.

## Formula
Simple Interest = (Principal x Rate x Time) / 100

## Example
- Principal = 1000
- Rate = 5%
- Time = 2 years
- Simple Interest = 100

## Usage
1. Enter principal amount
2. Enter rate of interest
3. Enter time period
4. Get simple interest result
